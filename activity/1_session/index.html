Ejercicio 5

  • ¿Cuales son las diferencias entre ejecutar un fichero JavaScript en un navegador web en formato <script> y en formato <script type=”module”>?. Busca documentación para enumerar los principales comportamientos de ambas.

    Respuesta de Jose Zuniga:

  • Principales caracteristicas:

    1. Permite dividir el javascript segun funcionalidad.
    2. Puede utilizar una extension .mjs
    3. La ejecucion de un archivo script tradicional es que al momento de cargar la pagina, el browser toma dicho script y lo ejecuta de principio a fin
    4. Mientras que un type 'module' permite a los navegadores modernos ejecutar codigo segun demanda, tal y como lo hace NodeJs (por ejemplo).
    5. Un modulo puede indicar que variables o funciones pueden ser accesibles fuera de dicho modulo. Lo cual es util para uso global
    6. Some references:
      1. type=module
      2. Modules MDN
